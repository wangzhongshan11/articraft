# 拉伸棱柱体

本示例演示如何用**拉伸**由二维轮廓得到棱柱实体。CadQuery 在每次绘图操作后，会把上一图元的中心压入栈顶，作为下一步的参考点；因此本例中先画圆再画矩形时，`rect()` 会以圆心为中心绘制，最后整体沿法向挤出固定厚度。

**要点：**

- 工作平面使用 `"front"`（前视），链式调用 `circle` → `rect` → `extrude`。
- 默认情况下，矩形与圆都会相对**当前工作点**居中，无需手动平移轮廓。
- 挤出距离 `0.5` 与轮廓尺寸共同决定最终包络。

```python
result = cq.Workplane("front").circle(2.0).rect(0.5, 0.75).extrude(0.5)
```
